<h1>Resources</h1>
<ul>
<li><a href="https://dropoff.startribune.com/">Strib Dropoff FTP server</a></li>
<li><a href="http://www.cometdocs.com/" target="_blank">Cometdocs </a>(for converting PDFs)</li>
<li>Star Tribune Slack -- Join the #data channel; most often the chatter on here is sharing and
discussing interesting data-driven work from other news organizations. If you aren't on Slack, here are directions.</li>
<li>Once you are on Slack, you might want to see if there are any channels in the <a href="https://newsnerdery.slack.com/">News Nerdery Slack</a> ecosphere that you might want to join. For example, there's a "helpme" channel
that is open to everyone; there is also a closed channel for women in data and digital journalism (Nerdettes);
and many others that have specific missions.</li>
<li><a href="https://www.ire.org/resource-center/listservs/subscribe-nicar-l/">NICAR Listserv</a>. Run by Investigative Reporters and Editors, this old-school email listserv is still a key place that data journalists ask questions (and get answers) to solve problems they've encountered. Some people also share links to data-driven stories they've published. You don't need to be an IRE member to join the listserv, but you do in order to search the archives.</li>
<li><a href="https://www.ire.org/blog/extra-extra/">IRE's Extra! Extra! Blog</a> This highlights data-driven and investigative stories published by news organizations around the world. Great source of inspiration. </li>
<li><a href="http://mjwebster.github.io/DataJ/">All of MaryJo's training materials.</a></li>
<li><a href="http://blog.chryswu.com/2018/01/23/nicar18-slides-links-tutorials/">NICAR 2018 links to tipsheets, slides, etc.</a> This is a repository compiled each year by a New York Times data journalist of all the tipsheets, slides and other links that are shared at the annual data journalism conference held by IRE each year.</li>
<li><a href="https://visitor.r20.constantcontact.com/manage/optin?v=001jNS0O4Ui3OO7md-9Ryd0WOKdq14U-VfK9aIRH18MLku7VRyaaHESUptkwHw-8FO3X8Dhpw6_U4bO-hrpYrIzmYZy_m-F01qUfYYiFg0mDpo%3D">Subscribe to Local Matters newsletter</a></li>
</ul>
<h1>Other options beyond Excel</h1>
<ul>
<li>SQL (structured query language) based programs such as Microsoft Access, mySQL, SQLLite. These allow you to work with much larger datasets and they are better equipped for joining datasets (like we did with VLOOKUP). Microsoft Access is on your computer but it has more limitations than other options. Materials for learning Access are on my <a href="http://mjwebster.github.io/DataJ/">data training materials website.</a> If you want to try one of the other options, I can help you get the necessary software installed, etc.</li>
<li>GIS (geographic information systems) software allows you to look for geographic patterns in your data. The free, open-source software <a href="https://qgis.org/en/site/">QGIS</a> is used by several people in the newsroom and there are tutorials and directions online that make it relatively easy to learn.</li>
<li> <a href="https://www.r-project.org/">The programming language called R</a> is growing in popularity in data journalism and is the most widely used tool in academia and the larger data science community. Using the open-source interface called <a href="https://www.rstudio.com/">RStudio</a> makes it seem more like a tool rather than a programming language. You can import/process/clean data, then do analysis and make charts and even publish web pages with your results. I've been using it now for about a year and find it's increasingly becoming my "go-to" tool. I can point you to quite a few online tutorials written by fellow journalists, and a data journalist just published a textbook on learning R. And I'd be glad to hold some beginner training sessions in the newsroom, as well.</li>
<li>There are other programming languages out there as well. Another very popular one in the data journalism world is Python, which is useful for scraping websites, writing scripts to automate data cleaning and analysis. Several members of our data team have Python skills. Some data journalists use Python with an interface called <a href="http://jupyter.org/">Jupyter Notebook</a>, which allows you to see your code and your results in line with each other and to more easily share your results with others.</li>
</ul>
