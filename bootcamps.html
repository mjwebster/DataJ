Data Journalism bootcamp sessions

Spring 2018

This is the schedule and class materials for a series of bootcamps available to University of Minnesota journalism students in spring semester 2018. Advanced registration required.

Schedule

Friday, Feb. 2, 8 a.m. to 10 a.m.

Interviewing data, part 1: How to find the right source, how to prepare for your interview, common questions that you'd ask almost any dataset, and how to ask basic questions using Microsoft Excel spreadsheets. You'll learn how to use basic formulas and functions, sorting, filtering and PivotTables.

Friday, Feb. 9, 8 a.m. to 10 a.m.

Interviewing data, part 2: We'll continue asking questions, getting into more complicated datasets and harder questions. This session will continue to use Excel, getting further practice in the skills covered in part 1, and adding on a few more useful skills.

Friday, Feb. 16, 8 a.m. to 10 a.m.

Dealing with dirty data: It's almost impossible to do a data analysis without first having to deal with problems that data comes with. We'll talk about the common scenarios you might encounter -- ranging from getting data out of PDFs to standardizing inconsistent data values to rearranging badly structured data. We'll use Excel, OpenRefine and an online tool for getting data tables out of PDFs.

Friday, Feb. 23, 8 a.m. to 10 a.m.

Intro to R using RStudio: One of the hot coding languages in both data journalism and the larger data science community is called R. A great advantage to using a coding language, such as R, for your data analysis is that you end up with reproducible results. The code you wrote for the analysis can easily be repeated, or reviewed by someone else. Or perhaps you need to make a slight change and go back and re-run the whole thing. You can't do that in spreadsheets. This session will introduce you to some basic data analysis and data visualization techniques using R and leave you with suggestions for how you can continue expanding these skills on your own.

Friday, March. 2, 8 a.m. to 10 a.m.

Turning your data analysis into visualizations: We'll take results of some of our earlier analyses to make charts and maps for online publication using readily-available tools. In addition to learning these tools, we'll cover best practices in visualization design and how to choose the right chart or map type.
